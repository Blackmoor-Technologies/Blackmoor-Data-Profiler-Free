# Blackmoor Data Profiler Product Boundary

## Free Edition

Blackmoor Data Profiler Free is the local-first version of the product.

It provides a practical CSV/Excel intake workflow:

**Inspect → Profile → Flag Issues → Document → Export**

## Free Edition Includes

- CSV upload
- Excel upload
- Excel sheet selection
- Sample dataset selector
- Dataset overview
- Column profile
- Data quality findings
- Deterministic data dictionary placeholder
- Markdown report export
- Generated sample reports
- Screenshots and demo materials
- Local setup, run, and validation scripts
- Friendly file-loading error handling
- Local-first and privacy-safe usage notes
- Fictional sample data only

## Free Edition Does Not Include

- AI-generated column descriptions
- AI-generated cleaning recommendations
- SQL CREATE TABLE generation
- Power BI or dashboard readiness reports
- PDF export
- HTML export
- Branded or custom report templates
- Bulk folder profiling
- Multi-file comparison
- Schema drift detection
- Saved profile history
- Project workspaces
- Recurring file checks
- Quality scoring
- Data contract or expected-schema rules

## Future Pro / Paid Feature Candidates

These features may be considered for a future paid or Pro edition.

- AI-assisted data dictionary descriptions
- AI-generated cleaning recommendations
- SQL CREATE TABLE generator
- Power BI readiness checklist/report
- Dashboard readiness scoring
- PDF report export
- HTML report export
- Branded report templates
- Bulk folder profiling
- Multi-file comparison
- Schema drift detection
- Saved profile history
- Project workspaces
- Recurring file checks
- Quality scoring
- Expected schema / data contract rules

## Boundary Note

Future Pro candidates should not be added to the Free Edition unless there is a deliberate product decision to move them into the free tier.
