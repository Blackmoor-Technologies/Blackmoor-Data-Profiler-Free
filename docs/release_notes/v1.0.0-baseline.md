# Blackmoor Data Profiler v1.0.0 Baseline

## Summary

This baseline release delivers a local-first Streamlit profiler for CSV and Excel files with practical dataset visibility and Markdown reporting.

## Included Capabilities

- Local CSV and Excel upload
- Excel sheet selection
- Dataset Overview display
- Column Profile display
- Data Quality Findings display
- Markdown report download
- Sample dataset selector
- Generated example reports under `reports/examples/`
- Demo screenshots under `docs/screenshots/`
- Friendly file-loading errors for malformed CSV/Excel input
- Sidebar app version label (`v1.0.0-baseline`)

## Validation

- Run `scripts\validate.bat`
- Current passing test count: 34 tests

## Known Limitations

- Quality checks are heuristic and intended as first-pass indicators
- No PDF or HTML export
- No AI/API integration
- No saved report history
- Custom dark UI treatment is deferred
- This is an early local-tool baseline

## Local-First / Sample Data Notes

- The app is designed for local-first workflows
- Files are processed in the locally running app session
- Sample files in `sample_data/` are fictional and suitable for demo/testing
- Use fictional or authorized data and follow your organization policies
