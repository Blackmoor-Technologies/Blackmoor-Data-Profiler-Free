# Blackmoor Data Profiler v1.1.0 Free Edition

## Summary

Blackmoor Data Profiler Free Edition is a local-first CSV and Excel profiling tool for analysts, builders, and small teams.

This release packages the current profiler as a clear Free Edition baseline. It helps users inspect datasets, understand column structure, identify common data quality issues, generate a basic data dictionary, and export Markdown profile reports.

## Included Capabilities

- CSV upload
- Excel upload with sheet selection
- Built-in fictional sample datasets
- Dataset overview
- Column profile
- Data quality findings
- Basic deterministic data dictionary placeholder
- Markdown report export
- Generated example reports
- Screenshots and demo materials
- Local setup script
- Local run script
- Local validation script
- Friendly file-loading error handling
- Local-first usage and privacy notes

## Data Quality Checks

The Free Edition includes deterministic first-pass checks for:

- Duplicate rows
- Blank-heavy columns
- Constant columns
- Possible ID columns
- High-variety text columns
- Possible date columns
- Suspicious mixed-type columns

These checks are intended as helpful indicators, not final data quality judgments.

## Local-First Notes

This project is designed to run locally.

Uploaded files are processed by the locally running Streamlit app session and are not intentionally sent to external services by this project.

Users should still follow their organization’s data handling policies and should avoid using sensitive, regulated, confidential, proprietary, or employer-owned data unless they are authorized to do so.

## Free Edition Boundary

The Free Edition focuses on:

**Inspect → Profile → Flag Issues → Document → Export**

The Free Edition does not include higher-value Pro candidate features such as:

- AI-generated data dictionary descriptions
- AI-generated cleaning recommendations
- SQL CREATE TABLE generation
- Power BI or dashboard readiness reports
- PDF or HTML export
- Branded report templates
- Bulk folder profiling
- Multi-file comparison
- Schema drift detection
- Saved profile history
- Project workspaces
- Recurring checks
- Quality scoring
- Expected schema or data contract rules

See `PRODUCT_BOUNDARY.md` for the full Free vs Pro boundary.

## Known Limitations

- Quality checks are heuristic and may require human review.
- Date detection and mixed-type detection are first-pass indicators.
- Data dictionary descriptions are placeholders only.
- No PDF or HTML export is included.
- No AI/API integration is included.
- No saved report history is included.
- No bulk folder profiling or multi-file comparison is included.
- Custom dark UI treatment is deferred.

## Status

This release represents the packaged Free Edition baseline for Blackmoor Data Profiler.

Future work should avoid adding Pro candidate features to the Free Edition unless there is a deliberate product decision to move them into the free tier.
